# Smarter-Last-Mile-Logistics-An-Autonomous-Transport-Robot-with-Enhanced-IoT-Features-
We present a hybrid navigation system that fuses global IoT-guided re-routing with local, sensor-driven obstacle avoidance. The global planner continuously updates a time-varying graph with congestion and risk signals (weighted by α,β,γ), while the local layer enforces collision-free motion.
